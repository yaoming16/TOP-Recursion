# Recursion Project

This project contains JavaScript implementations of recursive algorithms as part of [The Odin Project - Recursion lesson](https://www.theodinproject.com/lessons/javascript-recursion).

## Implementations

- **fibonacci.js** - Fibonacci sequence generator using recursion
- **mergeSort.js** - Merge sort algorithm implementation

## About

Created by [yaoming16](https://github.com/yaoming16)

Portfolio: [yaoming16.github.io](https://yaoming16.github.io/)

## Usage

Run the scripts with Node.js:

```bash
node fibonacci.js
node mergeSort.js
```
